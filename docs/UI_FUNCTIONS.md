# 主界面功能说明

## 视频区域

| 控件 | 功能 |
| --- | --- |
| 原始画面 | 显示摄像头、视频文件或 RTSP 的原始输入 |
| 检测画面 | 显示病害框、类别和置信度 |
| 横向检测 | 原始画面与检测画面左右排列 |
| 纵向检测 | 原始画面与检测画面上下排列 |

## 输入与运行控制

| 控件 | 功能 |
| --- | --- |
| 文件 | 选择本地图片或视频 |
| 摄像头 | 选择本机摄像头 |
| RTSP | 输入网络视频流地址 |
| 运行/继续 | 启动检测，或从暂停状态继续 |
| 停止 | 停止当前检测和推流任务 |
| 模型选择 | 选择 `pt/` 目录中的 YOLO 权重 |
| 推流分辨率 | 选择跟随源画面、360p、480p、720p 或 1080p；保持原始宽高比并自动补边 |
| 录制 | 选择录制原始摄像头画面或整个程序窗口，再次点击停止 |

录像保存到 `result/recordings/`。摄像头录像只保存原始输入画面；界面录像保存当前程序窗口。

## 实时遥测面板

| 字段 | 功能与来源 |
| --- | --- |
| 数据来源圆点 | 绿色表示真实数据，黄色表示虚拟、估算或混合数据，灰色表示等待或来源未知 |
| GPS 坐标 | 优先显示统一电控遥测位置，必要时兼容旧 OPGPS |
| 机器人状态 | 电控通用状态：待机、巡检、到树、充电、暂停或故障 |
| 速度 | 电控速度优先；速度字段缺失时才显示 GPS 估算速度 |
| 方位 | 电控上报的机器人行进方位角 |
| 相机高度 | 电控上报的相机视点高度 |
| 电池 | 电控上报的 SOC 和电池电压 |
| 故障码 | 电控故障码，0 表示无故障 |
| 果树 | 同时显示当前、左、右果树编号；数值补足四位，三个编号均为 0 时显示“当前无树” |
| 路线位置 | 当前路线编号和路径点编号 |
| 任务模式 | YOLO/原始推流方式，以及 real/debug/simulation 数据模式 |
| 当前帧 | 视觉主机当前处理帧序号 |
| 识别目标 | 当前帧检测到的病害目标总数 |
| 数据链路 | 当前启用的 HTTP、UDP、RTMP、统一遥测和旧 GPS 通道 |

鼠标悬停在实时字段名称或值上，可以查看对应字段用途。不可用字段显示 `—`，数值 0 会正常显示，不会被当成缺失值。
